# Visual Direction Brief

## Product Character
- Brand feeling: warm guidance with calm utility
- Trust posture: humane, competent, discreet
- Energy level: low-friction and present, never noisy
- Keywords: companion, reflection, continuity, confidence

## Visual Thesis
- Personality should feel like a thoughtful personal workspace, not a cold
  admin panel or sci-fi robot console.

## Surface Notes
- Mobile: conversation-first, reachable controls, emotional clarity
- Tablet: balanced split between transcript and contextual tools
- Desktop: focused chat plus supportive side panels for memory, tools, and
  settings

## Explicit Avoids
- robotic neon assistant aesthetics
- cluttered chatbot layouts
- sterile enterprise settings screens
