# Chat Canonical 100-Slice Closure Map

## Purpose

This document turns the `chat` flagship route into a single-surface execution
map with 100 bounded slices.

It exists to prevent broad, ambiguous polishing and to keep the route moving
toward `docs/ux/assets/aion-chat-canonical-reference-v4.png` in governed,
pixel-close steps.

## Governing Rules

- One active route surface at a time.
- Do not reopen `home`, `sidebar`, `dashboard`, or `personality` while the
  active `chat` closure lane remains below the current parity gate.
- Treat the canonical image plus explicit user notes as the merged spec.
- Stop early if the route clears the parity gate before consuming the whole
  provisional batch.
- If visible drift remains after a bounded batch, open the next bounded batch
  instead of silently widening scope.
